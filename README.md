# MA IPTV M3U

A free, open-source M3U playlist for Moroccan IPTV channels. Stream popular Moroccan TV stations directly in any M3U-compatible player.

## Channels

| Channel | Description |
|---------|-------------|
| **2M Tv** | Morocco's leading public broadcaster |
| **Al Aoula** | National public television |
| **Medi1 Tv** | Pan-Maghreb news and entertainment |
| **Arryadia HD** | Sports channel |

## Usage

### Option 1: Direct URL

Use this playlist URL in your IPTV player:

```
https://raw.githubusercontent.com/YubieTheSAVAGE/MA_IPTV_M3U/main/ma.m3u8
```

### Option 2: Local File

1. Download or clone this repository
2. Open `ma.m3u8` in your preferred IPTV/streaming app

### Compatible Players

- **VLC Media Player** — File → Open Network Stream → paste playlist URL
- **IPTV apps** — Tivimate, GSE Smart IPTV, IPTV Smarters, etc.
- **Kodi** — PVR IPTV Simple Client add-on
- **ffmpeg** — `ffplay ma.m3u8`

## Requirements

- Internet connection
- M3U/M3U8 compatible media player
- Streams may have geographic restrictions
